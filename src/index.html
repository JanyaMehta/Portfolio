Robofriends

RoboFriends is a ReactJS responsive single page web application, where the user can search for robot friends among all. It allow users to filter the Robots results data via the search form. It is Developed starting from the create-react-app package, the codebase is crafted in JavaScript/JSX. The application connects to a mock API from JSON Placeholder through a Fetch statement,allowing for real-time data interaction. Styling is implemented using the Tachyons NPM module, known for its simplicity and efficiency in applying styles using custom abbreviated CSS classes. RoboHash injects personality into the robot friends by generating unique and visually appealing robot avatars.
